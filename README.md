# stars-and-stories

A collection of small projects about stars and stories.

## Projects

- **[star-projector-simulator/](star-projector-simulator/)** — a 3D simulator
  and design tool for a homemade iPhone-flashlight star projector. Tune the star
  layout, cap shape, beam field-of-view and optical effects with sliders, then
  export a printable STL of the 3D-printed cap. Single self-contained HTML file,
  no build step.
